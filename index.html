jquery.DateTimePicker

The design and functionality was inspired by Schlu.

I had the need for all three varieities:

  • A DateTime Picker
  • A Date Picker
  • A Time Picker

The idea was to be simple, not flashy and styleable.

I use SASS, but I've included the generated CSS in the repo as well.

Dependencies

  • jQuery UI
  • FlyDOM

Known Issues

  • The height of the date picker changes weirdly as you change months.
